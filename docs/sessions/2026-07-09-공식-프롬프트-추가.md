# 공식 프롬프트 자산 추가

## 목적
`prompts/`를 AssetTrail 공식 프로젝트 자산으로 두고, 내보내기 JSON을 활용한 포트폴리오·은퇴 점검 프롬프트를 추가한다.

## 핵심 결정
- `prompts/`는 개인 임시 메모가 아니라 재사용 가능한 공식 자산으로 관리한다.
- 프롬프트는 투자 추천보다 자산배분, 의사결정, 은퇴 가정 점검에 초점을 둔다.
- 데이터에 없는 현재가, 전망, 세금, 건강보험 같은 값은 추측하지 않고 확인 질문으로 남기게 한다.

## 변경
- `prompts/포트폴리오-리뷰.md` 추가.
- `prompts/은퇴가정-점검.md` 추가.
- `prompts/README.md` 목록에 두 프롬프트를 추가.
- `docs/HANDOFF_NOTES.md`에 작업 결과와 검증을 반영.

## 검증
- `npm run check:js` 통과.
- 앱 코드 변경이 아니라 전체 테스트는 실행하지 않았다.
