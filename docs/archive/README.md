이 폴더는 현재 진입 문서에서 대체된 상세 과거 문서를 보관한다.

## 보관 원칙

- 오래된 문서는 삭제하지 않고 이 폴더로 옮긴다.
- 과거 결정, 구현 맥락, 세부 할 일 목록을 추적할 때 참고한다.
- 현재 작업의 기준은 루트의 `AGENTS.md`와 `docs/`의 활성 문서다.
- 활성 문서가 새로 생겨 기존 문서를 대체하면, 기존 문서를 이 폴더로 옮기고 참조 경로를 갱신한다.
